# 🧩 Web Snippets

A collection of reusable React components, custom hooks, and CSS variables.

## 🎨 Highlights

- **UI Components**: Modal, Button, Card, and Form elements.
- **Custom Hooks**: `useLocalStorage`, `useFetch`, and `useAuth`.
- **Utilities**: Debounce, throttle, and date formatting.
- **Styles**: Modern CSS/SCSS theme variables.

## 💻 Usage

```javascript
import { Button } from './components/Button';
import { useLocalStorage } from './hooks/useLocalStorage';
```
